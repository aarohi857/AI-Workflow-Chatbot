# 🎨 Frontend Enhancements

## UI/UX Improvements
- [ ] **Dynamic Themes**: Allow users to create custom color palettes beyond just Purple/Cyan.
- [ ] **Voice Synthesis**: Add text-to-speech for Aarvix using Web Speech API.
- [ ] **Terminal Mode**: A secondary UI mode that looks like a pure Linux terminal.

## Charts & Data
- [ ] **Live Data Streams**: Support for updating charts in real-time as data comes in.
- [ ] **3D Graphs**: Support for Plotly 3D scatter and surface plots for Zenvix.
- [ ] **Image Generation UI**: A dedicated panel for DALL-E or Stable Diffusion generation.

## Components
- [ ] **File Explorer**: A sidebar component to see uploaded files and their structure.
- [ ] **Session History Search**: Search bar within the chat history sidebar.
